API 설정

지도를 활성화하기 위해서는 Kakao Map JavaScript API Key 등록이 필요합니다. 키는 브라우저의 LocalStorage에 안전하게 저장됩니다.

* 카카오 개발자 콘솔에서 플랫폼 도메인(예: http://localhost:포트 또는 http://127.0.0.1:포트)을 등록해야 지도가 동작합니다.
* 기본적으로 제공된 키가 사용되지만, 한도 초과 시 개인 키를 입력할 수 있습니다.
